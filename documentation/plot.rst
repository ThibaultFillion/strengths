Plot
====

The strengths.pyplot module provide a few conveinient plotting functions to visualization.
Those function rely on the Matplotlib package (Hunter, 2007) [#matplotlib]_, as they act as convenient wrapper
around function such as matplotlib.pyplot.plot, matplotlib.pyplot.imshow or matplotlib.pyplot.colorbar.
